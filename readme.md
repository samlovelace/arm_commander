# Robot Commander 
The Robot Commander is a command line tool for sending commands to various subsystems

## Installation 
Clone into the robot_ws with the folder structure as

robot_ws
  - src
    - robot_idl
    - robot_commander

## Building 
Compile the arm workspace 
```bash
$ colcon build 
```
