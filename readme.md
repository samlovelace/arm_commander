# ARM Commander 
The Arm Commander is a tool for sending waypoints and other commands to the ARM module

## Installation 
Clone the arm repo from https://github.com/samlovelace/arm.git and run the clone.sh script 

## Building 
Compile the arm workspace 
```bash
$ colcon build 
``` 

## Usage 

### Supported Commands 
 - enable
 - waypoint
